# Task 3 - Dashboard Design

## Tool Used: Power BI
## Dataset: Superstore Sales Dataset

## Dashboard Features:
- 4 KPI Cards (Sales, Profit, Orders, Discount)
- 3 Slicers (Year, Region, Category)
- 3 Interactive Charts
- Consistent Color Theme

## Charts Created:
1. Sum of Sales by Year - Line Chart
2. Sum of Profit by Region - Pie Chart
3. Sum of Sales by Category - Bar Chart

## Key Insights:
- Technology highest sales category
- West region best performance
- 2023 highest sales year
- High discount = Low profit

- Power BI Dashboard
